# 🎓 Student Marks Manager

A modern and efficient Student Marks Management System built to manage student records, subjects, marks, and academic performance efficiently. This project demonstrates concepts of database management, CRUD operations, validation, modular programming, and backend application architecture.

---

## 🚀 Features

- Add Student Records
- Update Student Details
- Delete Student Records
- Add Subject Marks
- Calculate Total Marks
- Calculate Average Marks
- Grade Calculation
- Search Student Records
- Input Validation
- Database Integration
- Modular Code Structure

---

## 🛠️ Tech Stack

- Python
- SQLite / MySQL
- File Handling
- Object-Oriented Programming
- Database Queries

## 📌 Module Overview

### `main.py`
Acts as the main entry point of the application.

### `students.py`
Handles student registration, updates, deletion, and management.

### `marks.py`
Manages subject marks, calculations, and grading logic.

### `database.py`
Responsible for database connection, queries, and secure data storage.

### `validations.py`
Contains input validation and error checking functions.

### `utils.py`
Reusable helper functions used throughout the project.

### `reports.py`
Generates reports, summaries, and academic performance analysis.

## ⚡Installation

Clone the repository:

```bash
git clone https://github.com/student-marks-manager/student-marks-manager.git
```

Move into the project directory:

```bash
cd student-marks-manager
```

Run the project:

```bash
python main.py
```
